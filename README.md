# FocusBoard

勉強中に表示しておける時計・ポモドーロPWAです。サーバーやアカウントは不要で、設定とタイマー状態は端末内だけに保存されます。

## 機能

- サムネイルから固定背景または自動切替を選べる、フェード式の背景スライドショー
- 端末内の任意画像を最大8枚までIndexedDBへ保存し、背景として選択・削除
- 午前・午後表記を省いた12時間表示／24時間表示、秒、サイズ、フォント、配置を変更できる時計
- 時計・日付とタイマーの色を別々に設定でき、背景解析による自動調整にも対応。自動調整では時計周辺の背景サンプルのうち、最も低いコントラストを基準に色を選択
- 背景画像を専用の編集モードでドラッグ・ピンチ・ホイール操作できる背景トリミング設定。編集中も時計を確認でき、背景の移動・拡大縮小の操作中だけ時計を一時的に隠す。時計をタップすると色などの表示設定を変更可能
- スカイ、ラベンダー、ミント、ピーチ、ローズのパステルテーマと、時計の色・タイマーの色を個別指定できるカスタムテーマ
- `yyyy/mm/dd 曜日`、`mm/dd 曜日`などのプリセットと、記号を組み合わせたカスタム日付形式
- 作業、短い休憩、長い休憩に対応したポモドーロ
- 実施中／休憩を分けられる通常カウントダウン・カウントアップ
- 開始後に円形へ縮小し、円周進捗を表示するドラッグ可能なフローティングタイマー
- フローティングタイマーはクリック／タップで、背景を邪魔しにくい円形ミニ表示へ切り替え可能
- 設定カード・円形タイマー・収納ボタンに共通で適用できる背景不透明度調整
- 開始前の設定カードも左端のコンパクトなボタンへ収納・復元
- `endAt`基準で、画面を閉じた後や再読み込み後もずれにくいタイマー
- 時計・日付の色は初期状態を自動調整とし、背景上のプレビューを見ながら白／黒／テーマ色／カスタム色へ切り替え可能。カラーコード入力は詳細設定に収納
- iPadの縦向き・横向き、safe area、タッチ操作に対応した設定ドロワー。iPhone、Android、PCでも同じ基本操作を利用可能
- 背景をタップすると画面右下に表示される設定ボタン。設定は「背景」「表示」「タイマー」「保存・リセット」の4カテゴリーに整理し、詳細項目は折りたたんで表示
- スマホでは下から開く設定シート、iPad・Macでは右側から開く設定パネル。数値入力付きスライダー、3×3配置グリッド、自動保存状態、直前の変更の取り消しに対応
- 設定から全画面表示を切り替えられ、時計・日付・タイマーのサイズは「小さめ／標準／大きめ」で調整できる
- 追加背景の保存枚数・使用容量表示、削除、並べ替えに対応
- 保存・リセットから設定をJSONファイルへエクスポートでき、アプリバージョンの確認とアプリの再読み込みもできる
- オフライン起動、設定初期化、タイマー削除、ローカルデータ削除、PWAキャッシュ削除

## 開発環境の準備

Node.js 20以降とnpmをインストールしてください。WindowsではPowerShellまたはWindows Terminalで次を実行します。

```bash
npm install
npm run dev
```

Viteが表示したURLをブラウザで開きます。同一ネットワークのiPadから確認するときは`npm run dev -- --host`を使用してください。

## テストとビルド

```bash
npm test
npm run build
npm run preview
```

成果物は`dist`に生成されます。通常のビルドは相対パスを使用するため、サブディレクトリからも配信できます。明示的な公開パスを使う場合は、Windows PowerShellで次のように指定します。

```powershell
$env:VITE_BASE_PATH="/repository-name/"
npm run build
```

## GitHub Pagesへの公開

1. このプロジェクトをGitHubリポジトリの`main`ブランチへpushします。
2. GitHubの **Settings → Pages** を開きます。
3. **Build and deployment** のSourceを **GitHub Actions** に設定します。
4. `main`へのpush、またはActions画面の手動実行で`.github/workflows/deploy.yml`がテストとビルドを行い、`dist`を公開します。
5. `https://<user>.github.io/<repository-name>/`へアクセスします。

Actionsではリポジトリ名から`VITE_BASE_PATH`を設定するため、Project PagesでもアセットとService Workerが正しい範囲に配置されます。ユーザーサイト用の`<user>.github.io`リポジトリでは、必要に応じてworkflowの値を`/`へ変更してください。

## iPadのホーム画面に追加

1. 公開URLをiPadのSafariで開きます。
2. 共有ボタンを押し、**ホーム画面に追加**を選びます。
3. 名前を確認して**追加**を押します。
4. 以後はホーム画面のFocusBoardアイコンからstandalone表示で起動できます。

初回読み込み後は主要ファイルがService Workerへキャッシュされ、オフラインでも起動できます。新しい版の反映にはオンライン状態で一度起動してください。

## 保存と初期化

設定は`focusboard:settings`、タイマーは`focusboard:timer`としてlocalStorageへ保存します。アップロードした背景画像は`focusboard-backgrounds`というIndexedDBへ端末内保存します。時計・日付の色、位置、自動調整は背景ファイルIDごと、タイマーの色は共通設定として保存します。時計・日付、開始前タイマー、開始後の円形タイマーの位置は縦向き／横向きごとに保存されるため、iPadを回転して元の向きへ戻しても位置がずれません。設定パネルの「設定する画面の向き」から、別の向きの配置も編集できます。時計とタイマーの背景自動調整はそれぞれ個別に切り替えられます。設定データはversionを持ち、不正データや保存機能が利用できない場合も初期値で起動します。旧設定の共有色と自動調整状態は、新しい時計・タイマー設定へ移行されます。

設定のエクスポートは「設定 → データ管理」から実行できます。出力ファイルには時計・背景・タイマーなどの設定だけが含まれ、端末内の背景画像ファイルと進行中のタイマー状態は含まれません。

背景画像の追加と選択は「設定 → 背景 → 背景を選ぶ・追加する」にまとめています。追加した画像や内蔵画像をクリック／タップすると、その画像の設定が開き、「この背景を調整」から専用の編集モードに入れます。編集モードはiPadのロック画面の壁紙編集を参考に、背景を大きく見せ、上部に小さなツールバーを表示します。背景を1本指またはマウスでドラッグすると移動、2本指のピンチやホイールで拡大縮小できます。編集中も時計は表示され、時計をタップすると時計・日付の色（自動調整／手動色）、表示、サイズなどをその場で変更できます。背景を移動・拡大縮小している間は時計を一時的に隠します。Apple Pencil、矢印キー、ツールバーのスライダーにも対応します。「中央に戻す」「変更を取り消す」で編集中の変更を整理できます。補助ツールバーは操作後すぐに収納され、右上の×（時計と重なる場合は空いている隅へ移動）からいつでも編集を終了できます。スマホでは全画面で操作し、編集モード中はブラウザ側のピンチズームとスクロールを抑止します。編集モード以外では背景操作レイヤーがタッチ操作を奪わず、ブラウザの通常操作を妨げません。

開始前のタイマー設定は、右上の収納ボタンから開始後と同じ円形タイマーへ切り替えられます。セットした時間を保ったまま再生でき、リセットで中央の設定カードへ戻ります。円を指やApple Pencilでドラッグすると位置を変更でき、キーボード利用時はフォーカス後に矢印キーでも移動できます。開始後は「一時停止」「リセット」を円形タイマーから操作できます。

背景をタップすると画面右下に「設定」ボタンが表示されます。ボタンから設定を開くと、スマホではボトムシート、iPadやMacでは右側のパネルとして表示されます。背景を最初に表示し、時計・日付などの表示設定、タイマー設定、保存・リセットをタブで分けています。細かい設定は各詳細項目から開きます。

開始後の円形タイマーは、タイマー本体のクリック／タップで時間だけの円形ミニ表示にできます。もう一度クリック／タップすると通常表示へ戻ります。

右下の設定ボタンにある **リセット / 削除** から、次を個別に実行できます。

- 設定だけを初期化
- タイマー状態だけを削除
- このアプリのローカルデータをすべて削除
- このPWAのキャッシュとService Workerを削除

完全削除する場合は、先にローカルデータを削除し、iPadホーム画面のFocusBoardを長押しして **ブックマークを削除** または **Appを削除** を選びます。必要に応じてiPadの **設定 → Safari → 詳細 → Webサイトデータ** から公開サイトのデータも削除してください。Webアプリからホーム画面のアイコン自体を削除することはできません。

初期画像は非表示ボタンで一覧と自動切替から外せます。追加した画像は削除ボタンから確認後に削除できます。
